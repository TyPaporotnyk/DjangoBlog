Houses Scraper(RealForeclose)

I created a RealForeclose scraper with many subsystems.
This scraper collects all data about auction house such as a house, an owner, contacts. Also this scraper collects a data about house sale status and price from Zillow, Realtor, Redfin and Chase, a crime risk and a school rating by house address. At the end scraper collects all given data to an excel file and puts at the end of the file a house picture. At the moment scraper works in the server and collect data around half of year without any problems.

Broadcaster

This project has been develop for EvaCodes.
In this project you can send messages to Telegram, WhatsApp and SMS register new Telegram accounts for mailing, write messages by templates, save templates, choose account for mailing and store all sended and received messages to databse.
This project have a web interface and backend side with 3 celery tasks for mailing.
EvaCodes company uses this project for inform them contacts. At the moment project hosts in their server and works perfectly.
